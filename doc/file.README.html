Description

In most servicenow environments the cmdb is used for registration of information about the node This plugin provides a flexible way to implement some of these attributes in the cmdb as facter facts.

Since not every CMDB is moddeled in the same manner, this facter plugin uses a yaml config file for connection to servicenow and an array of servicenow fields that must be converted to facts of the current running node.
